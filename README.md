# <機能>
* 標準入力から読み込んだ数字の和を出力する。
* 偶数か奇数かを判別する。
* 素数かを判別する。

* plusを保存した状態で以下のコマンドを入力すると、1から2までの数字を順番に読み込むことが出来る。
* $seq 2 | ./plus
